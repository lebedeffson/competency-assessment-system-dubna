# Система оценки надпрофессиональных компетенций студентов
Веб-система для диагностики надпрофессиональных компетенций (soft skills) студентов с построением личностной карты компетенций и формированием персонализированных рекомендаций по профессиональному развитию.

## Быстрый старт

### Запуск с Docker (рекомендуется)

Если у вас установлен Docker, самый простой способ запустить приложение:

```bash
# Запуск с Docker Compose
docker-compose up -d

# Приложение будет доступно на http://localhost:5000
```

### Установка (без Docker)

#### Шаг 1: Активируйте виртуальное окружение
```bash
source ~/venv/bin/activate
```

#### Шаг 2: Установите зависимости
```bash
pip install -r requirements.txt
```

#### Шаг 3: Инициализируйте базу данных
```bash
python init_db.py
```

#### Шаг 4: Запустите приложение
```bash
python app.py
```

Откройте браузер: **http://localhost:5000**
